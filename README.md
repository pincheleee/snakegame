# Python Snake Game

A classic Snake game implementation using Python and Pygame.

## Requirements
- Python 3.x
- Pygame

## Installation
1. Install the required dependencies:
```bash
pip install -r requirements.txt
```

## How to Play
Run the game using:
```bash
python snake_game.py
```

## Controls
- Use arrow keys (↑, ↓, ←, →) to control the snake's direction
- Press 'R' to restart the game when game over
- Close the window to quit the game

## Features
- Classic snake gameplay
- Score tracking
- Grid-based movement
- Game over detection
- Food spawning
- Visual grid for better visibility
